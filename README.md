# CEFET-MG Question Classifier

## About the Project

This project was created for educational purposes.

CEFET-MG Question Classifier is a Python-based application that uses Machine Learning and Natural Language Processing techniques to automatically classify questions from CEFET-MG entrance exams by subject and topic.

The goal of the project is to help students organize large collections of questions and create more efficient study routines.

## Features

- Extract questions from PDF files
- OCR support with Tesseract
- Automatic classification by subject and topic
- Organized folder structure
- Semantic similarity using embeddings
- Support for multiple subjects

## Technologies

- Python
- OpenCV
- PyTesseract
- Sentence Transformers
- PyTorch
- NumPy
- PDF2Image
